# Injector - Android SO 加载器

## 快速开始

### 要求
- Android Studio 2022.1+
- Java 17+
- Gradle 8.0+

### 编译步骤
```bash
./gradlew clean assembleDebug
```

### APK 位置
```
app/build/outputs/apk/debug/app-debug.apk
```

### 功能
- Shizuku ADB 权限集成
- Root 权限检测  
- SO 文件动态注入
- 实时日志输出
- Material Design UI

### 权限
- QUERY_ALL_PACKAGES
- USE_SHIZUKU
